# 🛒 Ecommerce SQL Data Analysis Project

## 📌 Project Overview

This project demonstrates SQL skills by analyzing an Ecommerce database using MySQL. The objective was to extract meaningful business insights using SQL queries, joins, aggregate functions, subqueries, views, and indexes.

The project covers customer analysis, product analysis, sales analysis, revenue analysis, and monthly performance reporting.

---

## 🎯 Objectives

* Retrieve and filter data using SQL queries
* Perform sales and revenue analysis
* Analyze customer purchasing behavior
* Analyze product performance
* Practice SQL joins and subqueries
* Create reusable SQL views
* Optimize database performance using indexes
